# Weekly Learning Prompt

command:: /weekly-learning
folder:: 02-Daily/Weekly-Learnings

## 用途

把一周里的日记、资料、项目推进和 AI 对话，整理成一份认知变化报告。

## 输入

```text
/weekly-learning
请读取本周 daily notes、项目笔记、Inbox 中未处理材料，输出本周学习与认知变化。
```

## 输出结构

- 我原来以为：
- 现在我更倾向于：
- 触发变化的证据：

## 2. 值得注意的第三方观点

- 观点：
- 来源：
- 我赞同/保留的部分：

## 3. 现实项目推进

- 项目：
- 本周推进：
- 阻塞点：
- 下周动作：

## 4. 下周 Todo

- [ ] 最重要的一件事
- [ ] 需要约人/等待反馈
- [ ] 需要补资料
