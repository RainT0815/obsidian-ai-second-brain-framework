# 新手手把手操作指南

## 第 0 步：确认 Claudian 能用

1. 打开 Obsidian。
2. 打开右侧 Claudian 面板。
3. 新建一个聊天 tab。
4. 模型选择保持 `Opus`。
5. 发送：

```text
你好，请告诉我你能否读取当前 vault。请先列出根目录有哪些文件夹。
```

如果它能列出 `00-Meta`、`02-Daily`、`wiki` 等目录，就说明连接正常。

## 第 1 步：向它介绍你是谁

打开并填写：

- [[00-Meta/Personal/给AI的自我介绍]]
- [[00-Meta/Personal/个人系统整理-AI创作者路线]]
- [[00-Meta/Personal/Life Statement]]

建议先写 10 分钟，不用追求完整。AI 需要的是“足够开始帮你”的上下文。

填写后，在 Claudian 里发送：

```text
/context
请读取：
- 00-Meta/Personal/给AI的自我介绍.md
- 00-Meta/Personal/个人系统整理-AI创作者路线.md
- 00-Meta/Personal/Life Statement.md
- 00-Meta/Personal/digital-self/数字分身采访官.md
- 00-Meta/System/context-prompt.md

然后生成一份“我是谁”的上下文摘要，保存到：
00-Meta/Personal/AI-Context-Summary.md

摘要要包括：
1. 我的身份和工作
2. 我的长期目标
3. 我最近关注的事情
4. 我的 AI 主线和每日小输出要求
5. 我的精力边界和过载风险
6. AI 帮我时应该注意什么
7. 你还需要追问我的 5 个问题
```

## 第 2 步：发送今天的 To-Do

打开：

- [[02-Daily/Todo/如何发送Todo给AI]]

照着里面的模板，把今天的事情粘贴给 Claudian。

最短版本是：

```text
/todo
今天我要做：
- 第一件事
- 第二件事
- 第三件事

请帮我整理优先级，拆成可执行步骤，并写入 02-Daily/Todo/今天.md。
```

## 第 3 步：每天怎么用

早上：

```text
/context
请读取我的个人上下文、最近 7 天 daily note、当前项目，告诉我今天应该聚焦什么。
```

接着：

```text
/todo
今天我脑子里有这些事：...
请整理成今日行动表。
```

晚上：

```text
请读取今天的 Todo 和 Journal，帮我做晚间复盘：
1. 完成了什么
2. 卡住了什么
3. 明天最小下一步
4. 有什么应该沉淀到 Resources 或 wiki
```

## 第 4 步：处理资料

如果你看到一篇文章、视频、PDF 或 AI 对话觉得有价值：

1. 把原始内容放进 `raw/sources/` 或 `01-Inbox/`。
2. 在 Claudian 里发送：

```text
/ingest raw/sources/文件名.md
请按 Karpathy Wiki Schema 摄取这个来源。
```

AI 应该会做：

- 创建 `wiki/来源/...`
- 更新 `wiki/index.md`
- 更新 `wiki/log.md`
- 必要时创建 `wiki/概念/...` 或 `wiki/实体/...`

## 第 5 步：每周检查

周末发送：

```text
/weekly-learning
请读取本周 daily notes、Todo、项目和 wiki/log.md，生成本周学习报告。
```

再发送：

```text
/lint
请检查这个 vault：
1. 有没有未处理 Inbox
2. 有没有孤岛笔记
3. wiki/index.md 是否过期
4. 有没有重复概念
5. 哪些项目缺少下一步
```
