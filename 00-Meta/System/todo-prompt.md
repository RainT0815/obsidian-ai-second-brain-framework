# Todo Prompt

command:: /todo
folder:: 02-Daily/Todo

## 用途

把今天的想法、任务和压力转成清晰的行动清单，并放入当天 daily note 的 `Todo` 子文件夹。

## 输入

直接用自然语言告诉 AI：

```text
/todo
今天我需要处理这些事情：
- ...
- ...
请帮我按重要性和紧急性整理，并生成可执行任务。
```

## 输出模板

# {{date}} Todo

- [ ] 今天最重要的一件事：
- [ ] 这段时间最重要的一件事：
