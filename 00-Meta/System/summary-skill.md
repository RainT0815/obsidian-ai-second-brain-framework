# Summary Skill

command:: /summary
target:: 05-Resources/summary

## 用途

把一次 AI 对话、一篇文章、一个视频或一组资料，整理成可长期复用的知识笔记。

## 触发方式

```text
/summary
请把刚才这段内容整理成 Obsidian 笔记：
1. 摘要
2. 关键概念
3. 可行动建议
4. 应该创建的双链
5. 应放入的文件夹
```

## 输出要求

- 保留来源链接和时间。
- 摘要放入 `05-Resources/summary`。
- 概念拆成独立笔记，放入 `05-Resources/concepts`。
- 如果出现任务，转入 `02-Daily/Todo` 或对应项目。
- 如果出现长期领域，链接到 `04-Areas`。
