# Commands Hub

## Daily

- [[todo-prompt]] - turn messy tasks into an action plan.
- [[context-prompt]] - load current identity, projects, and open loops.
- [[daily-review-prompt]] - nightly review: what got done, did I stay on main line, input/output balance, tomorrow's next step.
- [[00-Meta/Personal/给AI的自我介绍]] - the first personal context file to fill in.
- [[02-Daily/Todo/如何发送Todo给AI]] - copy-paste recipes for daily task capture.

## Weekly

- [[weekly-learning-prompt]] - synthesize weekly cognitive changes.
- [[lint-prompt]] - inspect vault health and structure.

## Knowledge

- [[summary-skill]] - convert raw input into summary, concepts, links, and actions.
- [[ingest-prompt]] - ingest one raw source into the maintained wiki.
- [[query-wiki-prompt]] - answer questions from the compiled wiki.
- [[karpathy-wiki-schema]] - schema for raw/wiki style knowledge maintenance.

## Recommended Flow

1. First read [[新手手把手操作指南]].
2. Fill [[00-Meta/Personal/给AI的自我介绍]].
3. Start the day with `/context`.
4. Use `/todo` after writing the daily note.
5. Use `/summary` whenever you finish reading, watching, or discussing something useful.
6. End the week with `/weekly-learning`.
7. Run `/lint` before major cleanup.
8. Use `/ingest` for serious source material that should enter the long-term wiki.
